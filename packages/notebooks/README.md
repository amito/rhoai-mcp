# rhoai-mcp-notebooks

Workbench (Kubeflow Notebook) management plugin for RHOAI MCP Server.

## Installation

```bash
pip install rhoai-mcp-notebooks
```

## Features

- Create, list, start, and stop workbenches
- Manage workbench resources and configurations
- Support for various notebook images (JupyterLab, VS Code, RStudio)
